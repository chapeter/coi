
  * Install PIP

  ```
  $ wget https://bootstrap.pypa.io/get-pip.py
  
  $ python get-pip.py
  ```
  * Install Dependencies

  ```
  cd coi
  pip install -r requirements.txt
  ```
* Run webapp
  
  ```
  $ python coi.py
   * Running on http://0.0.0.0:5000/ (Press CTRL+C to quit)
  ```
* Repeat on each server
